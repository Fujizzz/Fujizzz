## Featured Research Project

### [Multidimensional Comparative Evaluation of Human-Written and LLM-Generated Code](https://github.com/Fujizzz/Multidimensional-Comparative-Evaluation-of-Human-Written-and-LLM-Generated-Code)

My undergraduate thesis and a reproducible research artifact comparing human-written code with five LLM-based generation approaches on 3,655 RepoEval tasks. The evaluation covers correctness, executability, engineering quality, contextual coherence, and risk exposure.

**Methods:** DeepSeek-Coder, Qwen2.5-Coder, DeepSeek API, Repoformer, and mini-swe-agent<br>
**Tasks:** Function, line, and API completion<br>
**Artifact:** Generation pipelines, per-sample results, aggregate metrics, documentation, and automated checks
